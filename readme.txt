1) npm init -y
2) npm install express - для обработки запросов
3) npm install --save docxtemplater pizzip - для редактирования документа и создания сжатого буфера данных
4) npm install uuid - для создания GUID в названии сформированного документа для уникальности
5) npm install log4gs - для логирования
6) npm install multer - для обработки FormData, получаемой из другого сайта
5) node app.js - запуск приложения
